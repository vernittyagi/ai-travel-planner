# AI Travel Planner - PlanGo
## Features
* AI generated itineraries
* model used - Groq llama
* Trip storage to MongoDB 
* Node.js backend
* Frontend with Vite, React
* Pexel api for getting images for the destination
* Recently created trips from the database
* Trip share with copy link option
* Instant search suggestions for destination using Geoapify autocomplete API
